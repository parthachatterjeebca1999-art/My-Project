# Project Images

Place your project screenshots here with the following filenames:
- `project1.jpg` - E-Commerce Platform
- `project2.jpg` - Task Management App
- `project3.jpg` - Portfolio Website

Recommended specifications:
- Format: JPG or PNG
- Size: 600x400 pixels
- File size: < 300KB each
- Quality: High resolution

These images will be displayed in the Projects section with hover effects.

## How to add more projects:

1. Add new image files (project4.jpg, project5.jpg, etc.)
2. Edit `index.html` and duplicate a project card
3. Update the image source, title, and description
4. Update the project link in the "View Project" button
